📋 خطوات الحصول على API Key من Abstract API

  1. اذهب إلى Abstract API
  2. اضغط "Get Started" أو "Sign Up"
  3. سجل حساب جديد (مجاني)
  4. انتقل إلى Dashboard
  5. ابحث عن "Phone Validation API"
  6. انسخ API Key
  7. الصق API Key في التطبيق

💡 كيفية استخدام API Key

  1. افتح التطبيق
  2. في قسم "إعدادات API"، الصق API Key
  3. اضغط "حفظ API Key"
  4. ستظهر حالة "متصل" بجانب الحقل
  5. الآن عند تتبع أي رقم، ستحصل على بيانات حقيقية!
⚠️ ملاحظات مهمة:
  • لا تشارك API Key مع أي شخص
  • احتفظ بنسخة احتياطية من API Key
  • راقب استخدامك لتجنب تجاوز الحد المجاني
  • إذا نفدت الطلبات المجانية، يمكنك شراء خطة مدفوعة

🔧 استكشاف الأخطاء

إذا لم يعمل API:

  • تأكد من صحة API Key
  • تحقق من اتصال الإنترنت
  • تأكد من أن الرقم بصيغة صحيحة (+966xxxxxxxxx)
  • التطبيق سيعود للمحاكاة تلقائياً في حالة فشل API
